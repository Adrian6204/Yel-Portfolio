# Yel Bejoc Portfolio

This directory contains the source code for the personal portfolio of **Yel Bejoc**, a multimedia designer. The project is designed to showcase professional work through a modern, responsive digital interface, emphasizing a clean and vibrant aesthetic.

## Project Overview

The portfolio features a curated collection of selected design projects, a professional biography, and contact information. It is built with contemporary web standards to ensure optimal performance and a seamless user experience across all devices.

The design employs a **Glassmorphism** visual style, combining transparency and blur effects with a light, energetic color palette to reflect a creative and professional brand identity.

## Getting Started

To view this project on your local machine:

1.  **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) installed.
2.  **Installation**: Open a terminal in the project's root directory and install the required dependencies:
    ```bash
    npm install
    ```
3.  **Launch**: Start the development server:
    ```bash
    npm run dev
    ```

The application will be accessible via the local address provided in the terminal output (typically `http://localhost:5173`).
